# Premium Scrollytelling Portfolio

A high-performance, modern portfolio built with **Next.js 14**, **Framer Motion**, and **Tailwind CSS**.

## 🚀 Features

- **Smooth Scrollytelling**: Interactive canvas-based scroll animations.
- **Premium Design**: Dark-themed aesthetic with glassmorphism and custom animations.
- **Project Showcases**: Detailed case studies with cover photos and technical breakdowns.
- **Responsive Layout**: Optimized for all devices.
- **Interactive Skills & Education**: Visualized stats and timelines.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/)
- **Deployment**: [Vercel](https://vercel.com/)

## 📂 Structure

- `/src/components`: Reusable UI components (Projects, Skills, Navbar, etc.)
- `/src/app`: Next.js App Router pages and global styles.
- `/public`: Static assets (images, fonts, and scroll sequence).
